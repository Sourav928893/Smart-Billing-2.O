# 📱 Smart Billing 2.O

Smart Billing 2.O is a simple and efficient Android billing application designed for shop owners, small businesses, and freelancers. It helps users create invoices, manage customers, and track sales easily.

---

## 🚀 Key Features

✔ Create professional invoices  
✔ GST & Non-GST billing support  
✔ Manage customers and products  
✔ Track paid and unpaid invoices  
✔ Sales history and reports  
✔ Works completely offline  
✔ Secure local data storage
