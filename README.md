# Smart Billing 2.O

Smart Billing 2.O is a simple and efficient Android billing application designed for shop owners, small businesses, and freelancers. It helps users create invoices, manage customers, and track sales easily.

---

## 📱 Features

- Create professional invoices  
- Add and manage customers  
- GST and non-GST billing support  
- Track paid and unpaid invoices  
- View sales history  
- Manage products easily  
- Offline billing support  
- Simple and clean user interface  
- Data stored locally on device

## 🔐 Data Privacy

Smart Billing 2.O works completely offline.  
All data is stored locally on the user's device.  
No data is collected or shared with external servers.
